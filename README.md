## 🧠 Tech Stack

- **Frontend**: React, Tailwind CSS, Zustand, GSAP
- **Backend**: Express.js, Node.js, MongoDB
- **Storage & Deployment**: Cloudinary (media), Vercel (hosting)
- **AI Logic**: Custom role-based response engine (LLM-ready)
